WELCOME TO APPREDICT

Selamat datang di website Appredict. Appredict merupakan website yang digunakan untuk mempermudah perusahaan brand kosmetik Emina dalam menganalisis sentimen customernya selama menggunakan produk. Website ini dikembangkan dengan menggunakan 11000 data latih yang bersumber pada website Female Daily Review dengan Brand Emina serta algoritma yang digunakan adalah Algoritma 1D-Convolutional Neural Network. Model yang dibangun telah mencapai akurasi 80%.

Cara penggunaan aplikasi:

1. Silahkan masuk ke menu laman prediksi

2. Unggah file yang berisikan data dengan format file .csv

3. Klik "Submit", kemudian tunggu sampai muncul tabel hasil kelas klasifikasi sentimennya

4. Terdapat file hasil klasifikasi sentimen dalam bentuk .csv pada directory "Output"
